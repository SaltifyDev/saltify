# 内置插件

Saltify 内置了若干插件，可以选择安装。

## default-logging

**日志插件**，负责将运行时信息以统一格式输出到日志。

如需配置日志输出，请参考[日志实现](./logging)文档。

## command-help

**全局帮助指令插件**，注册一个 `help` 指令。

触发后，将以合并转发消息形式回复所有已注册指令的帮助文档，并按插件分组：

- 通过插件注册的指令归入对应插件组
- 未通过插件注册（直接调用 `client.command()`）的指令归入"其他指令"组

> [!NOTE]
>
> `/help` 指令本身也会出现在帮助文档中，归属于 `command-help` 插件组。
